<h1>🛒 E-Commerce Website</h1>

<p>
A full-stack E-Commerce Web Application built using the MERN stack.
Admin Panel (Admin can add/delete products),
Users can browse products, add/remove items to cart, authenticate securely
with hashed passwords.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>🔐 User Authentication (Signup / Login)</li>
  <li>Account create/delete</li>
  <li>🔒 Secure Password Hashing (bcrypt)</li>
  <li>🛍️ Product Listing</li>
  <li>🛒 Add to Cart & Remove from Cart</li>
  <li>📱 Fully Responsive UI</li>
  <li>Admin panel (add Product, delete Product)
</ul>

<hr>

<h2>📦 Languages and Tools</h2>

<h3>Frontend</h3>
<ul>
  <li>React.js (Vite)</li>
  <li>React Router DOM</li>
  <li>Axios</li>
</ul>

<h3>Backend</h3>
<ul>
  <li>Node.js</li>
  <li>Express.js</li>
  <li>dotenv</li>
  <li>cors</li>
  <li>bcrypt (Password Hashing)</li>
  <li>Mongoose</li>
</ul>

<h3>Database</h3>
<ul>
  <li>MongoDB (MongoDB Atlas)</li>
</ul>

<hr>

<h2>🚀 Deployment</h2>
<ul>

</ul>

<hr>

<h2>⚙️ Environment Variables</h2>

<h3>Backend (.env)</h3>

<pre>
PORT=5000
MONGO_URI=your_mongodb_connection_string
</pre>

<h3>Frontend (.env)</h3>

<pre>
VITE_API_URL=your_backend_url
VITE_ADMIN_ROLE=( Example Values : ADMIN , admin , MAINADMIN ) // defines the role value that will be recognized as an administrator in the application
</pre>


<hr>

<h2>💻 Installation</h2>

<h3>Clone Repository</h3>
<pre>
git clone https://github.com/nuthan-444/E-Commerce-Website
cd ecommerce
</pre>

<p>If you are beginner :</p>
<p><b>⚠️ Important:</b> Never push your <code>.env</code> file to GitHub.</p>

<h3>Backend Setup</h3>
<pre>
cd BackEnd
npm install
npm start
</pre>

<h3>Frontend Setup</h3>
<pre>
cd FrontEnd
npm install
npm run dev
</pre>
